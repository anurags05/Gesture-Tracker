# Gesture-Tracker

__Description:__
This Python based project demonstrates how to use computer vision techniques to track hand gestures and translate them into mouse movements. The project utilises OpenCV for image processing and machine learning algorithms for hand detection and recognition.

__Prerequisites:__
1) Python: 3.12 or newer installed.
2) Install pip python package manager.
3) OpenCV: Install via ```pip``` using ```pip install opencv-python```.
4) MediaPipe: Install using ```pip install mediapipe```.
5) PyAutoGUI: Install using ```pip install PyAutoGUI```.

__Installation and Running:__
1) Download project files into a location
2) Import project directory into IDE
3) Run ```app.py``` file.
